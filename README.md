# System Health Monitor

A lightweight Python tool for monitoring system resources (CPU, Memory, Disk, Network) with real-time alerts and logging.

## Features
- ✅ Real-time CPU, Memory, Disk, and Network monitoring
- ✅ Configurable alert thresholds
- ✅ Continuous or single-run monitoring
- ✅ Logging to file and console
- ✅ Command-line interface

## Installation
```bash
# Clone the repository
git clone https://github.com/NikhilPalliCode/system-health-project.git

# Install dependencies
